# AYS

## Please install the following packages:
#### $ npm init
#### $ npm install
#### $ npm install modules
#### $ npm install ejs
#### $ npm install express 
#### $ npm install body-parser

---------------------------------------------------------------

## Run the application in command line:
#### $ node app.js
#### Or 
#### $ nodemon start

server will start listening at port number 3001

Then open the following link
http://localhost:3001/
